# Supernaturaltabs

A modern single-page portfolio for Lite.

## Files

- `index.html` - main portfolio page
- `styles.css` - site styling
- `script.js` - minimal navigation interaction

## Deploying

1. Host the site on GitHub Pages, Netlify, Vercel, or any static host.
2. For GitHub Pages, push this repo to GitHub and enable Pages from the `main` branch root.
3. To preview locally, open `index.html` or run a local server:
   - `python -m http.server 8000`
   - visit `http://localhost:8000`
4. For a custom domain, register the domain and point it to your host with a CNAME or A record.
5. If using GitHub Pages with a custom domain, add a `CNAME` file containing the domain name.
